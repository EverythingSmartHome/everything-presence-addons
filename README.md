# Everything Presence Add-ons

Home Assistant add-ons for Everything Presence One/Lite products

## Enviroment variables

If running via a standalone docker container you can use the following

| variables | meaning |
| --------- | ------- |
| HA_URL | Full url used to access Home Assistant (eg. <http://homeassistant:8123>) |
| HA_TOKEN | A long liver Home Assistant access token |
| VERIFY_SSL | If set to true certificate verification will be disabled |
